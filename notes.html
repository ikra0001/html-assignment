HTML Tags

div Tag
a common tag used to contain multiple elements, and enables intuitive and logical division between groups of elements.
cite tag
used to create a citation of a source for the corresponding content like (e.g. quotes).
a tag
used for linking content and uses the "href" attribute to specify the location.
img tag
used to display images on a webpage, with the "src" attribute providing the location of the image, as well as the "alt" attribute to describe the content.
title tag
used to set a title for the webpage, and will be displayed on the top of a tab
